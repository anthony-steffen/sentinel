# Kit LinkedIn - Sentinel Fraud Platform

Este arquivo traz textos prontos para voce publicar o projeto no LinkedIn com foco em recrutadores de vaga junior/pleno.

## Objetivo da publicacao

1. Mostrar entrega tecnica real end-to-end
2. Demonstrar criterio de engenharia (arquitetura, seguranca, produto)
3. Facilitar teste do projeto por pessoas recrutadoras

## Post modelo (versao curta)

Texto sugerido:

```text
Acabei de finalizar o Sentinel Fraud Platform, um projeto full stack focado em monitoramento antifraude.

Stack:
- FastAPI + SQLAlchemy + Alembic
- React + TypeScript + TanStack Query
- PostgreSQL + Redis + RabbitMQ
- Docker + fluxo de demonstracao para recrutadores

Destaques:
- Dashboard analitico com KPI e graficos
- RBAC por perfil (ADMIN, ANALYST, OPERATOR)
- Audit logs e regras de seguranca
- Notificacoes em tempo real

Se quiser testar, posso enviar acesso de demo.
Feedback tecnico e muito bem-vindo.

#python #fastapi #reactjs #typescript #docker #postgresql #backend #frontend #portfolio #opentowork
```

## Post modelo (versao detalhada)

Texto sugerido:

```text
Projeto concluido: Sentinel Fraud Platform

Nas ultimas semanas desenvolvi uma plataforma full stack para monitoramento de operacoes com risco de fraude.

O que implementei:
- Pipeline de transacoes com score de risco e sinais antifraude
- Dashboard com visao executiva e operacional
- Controle de acesso por papeis (RBAC)
- Trilhas de auditoria para eventos criticos
- Realtime para notificacoes no front
- Infra com Docker para reproduzir ambiente rapidamente

Aprendizados que levo:
1) A importancia de modelar permissoes no backend e frontend juntos
2) Como manter observabilidade e seguranca sem perder produtividade
3) Como transformar um projeto tecnico em algo realmente testavel para recrutadores

Estou buscando minha primeira oportunidade na area e aberto a feedbacks tecnicos.
```

## Midia recomendada para o post

2. Post de bastidores tecnicos (dia 3)
3. Post de melhoria/iteracao (dia 7)
